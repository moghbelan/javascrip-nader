# JS-Variables-I
Upon cloning this repository, create an index.js file for your solution.

## Instructions
Please read and solve the following challenges below:

1. Create a variable for your first name and last name using let and assign your first and last name to the variables accordingly.
2. Using `let`, create a variable age and occupation and assign values to them.
3. Using `const`, create a gender and nationality and assign values to them.
4. Log the values of each variable on the console.
Change the values of each variable and log the values to the console again. What happens?

**NOTE**: Ensure you are using best variable naming practices
